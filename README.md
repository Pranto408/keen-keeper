# KeenKeeper

## 📌 Project Description
KeenKeeper is a smart relationship management web application that helps users track interactions with friends and maintain meaningful connections. It allows users to log calls, texts, and video interactions while keeping a timeline history for better communication tracking.

## 🚀 Live Demo
🔗 https://keen-keeper-52b9.vercel.app/

## 🛠️ Technologies Used
| Technology                            | Purpose                                                 |
| ------------------------------------- | ------------------------------------------------------- |
| React.js / Next.js                    | Build the user interface and overall frontend structure |
| React Router DOM / Next.js App Router | Handle page navigation and routing between pages        |
| Tailwind CSS + DaisyUI                | Styling, responsiveness, and reusable UI components     |
| Recharts                              | Data visualization and chart creation                   | 

## ✨ Key Features
1. **Interaction Tracking Timeline**  
   Users can log and view all interactions (Call, Text, Video) in a clean and organized timeline.

2. **Pie Chart Visualization**  
   Displays interaction statistics (Call, Text, Video) using a pie chart for better data understanding.

3. **Filter System**  
   Users can filter timeline activities by Call, Text, or Video to quickly find specific interactions.
